##########################################################
# monname.txt: lookup monster names
#
# This file contains a list of names for monster names.
# This is currently only used when one of your orcish apostles as
# Beogh's messiah gets named by Beogh, and for Hep's ancestor names.
# The dual-wielding coglin species also gets their weapon names from
# here.
#
# Some important orcish syllables
#   beogh, bog = referring to Beogh
#   ork, orc, org, og, ok etc. = referring to orcs
#
# Other syllables may be borrowed from real life, or made up.
#
# To determine a name for an orcish apostle, the game looks for this
# apostle's specific type, followed by the key word "name", e.g. "orc
# apostle warrior name" or "orc apostle wizard name". An apostle can get
# a name unique to its specific type or an ordinary orc name.
#
# Monster names support a small subset of what monster speech does:
# @RANDGEN@, @random_god@, @random_god_chaotic@, @random_god_evil@,
# @random_god_good@, @random_skill@, @random_skill_magic@, and
# @random_skill_mundane@.
#
# Feel free to add to this list. Like for the other database lists,
# you may add weights (defaults to w:10) to make certain names more
# or less likely.
#
##########################################################
