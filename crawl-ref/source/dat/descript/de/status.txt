Invis status

Ihr seid unsichtbar. Die meisten Feinde können Euch nicht sehen und erleiden
Abzüge für Treffer, obwohl sie Euren ungefähren Standort erraten können. Dies
führt mit der Zeit zu magischer Kontamination und kann gefährliche
Kontaminationsgrade verursachen, wenn Ihr bereits vor Erreichen dieses Status
kontaminiert wart. Wenn der Status ausgegraut ist, ist er aktiv (und verursacht
Kontamination), aber aus irgendeinem Grund wirkungslos (z. B. wenn Ihr vor
Kontamination glüht oder in seichtem Wasser steht).

sind resistent gegen Qualen.

Water status

{{
    local feat = view.feature_at(0, 0):gsub("_", " ")
    if you.flying() then
        if feat == "deep water" then
            return "Ihr fliegt über tiefes Wasser.
        elseif feat == "shallow water" then
            return "Ihr fliegt über seichtes Wasser.
        else
            return "Ihr fliegt über Wasser.
        end
    else
        local verb = you.race() == "Merfolk" and "schwimmt in" or "steht in"
        if feat == "deep water" then
            return "Ihr " .. verb .. " tiefem Wasser.
        elseif feat == "shallow water" then
            return "Ihr " .. verb .. " seichtem Wasser.
        else
            return "Ihr " .. verb .. " Wasser.
        end
    end
}}

Zot status

Ihr habt zu viel Zeit in einem Zweig verbracht, sodass das bösartige Wesen Zot
Euch bemerkt hat. Wenn Ihr die angegebene Anzahl von Runden in dem Zweig
verbringt, ohne in eine neue Ebene hinabzusteigen, wird Zot Euch finden und
auszehren.
%%%%
Zot count status

Wenn Ihr die angegebene Anzahl an Runden in diesem Zweig verbringt, ohne in eine
neue Eben hinabzusteigen, wird Zot Euch finden und auszehren.
%%%%
Curl status

Nach einem Angriff habt Ihr Euch reflexartig zusammengerollt, um Euch vor
weiteren Schlägen zu schützen. Ihr löst Euch jetzt aus dieser
Verteidigungshaltung und erhaltet keinen weiteren Schutz, bis Ihr erneut
getroffen werdet.
