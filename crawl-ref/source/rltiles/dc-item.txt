# A note on variations. To create additional variations for weapons or armour,
# use the %enchant_variation command e.g.:
#
#    weapon_normal WEP_SOME_WEAPON
#    %enchant_variation WEP_SOME_WEAPON shiny
#    weapon_shiny WEP_SOME_WEAPON_SHINY
#    %enchant_variation WEP_SOME_WEAPON runed
#    weapon_runed WEP_SOME_WEAPON_RUNED
#    %enchant_variation WEP_SOME_WEAPON glowing
#    weapon_glowing WEP_SOME_WEAPON_GLOWING
#    %enchant_variation WEP_SOME_WEAPON randart
#    weapon_randart WEP_SOME_WEAPON_RANDART
#
# You can also use the same tile for multiple variations e.g.:
#
#    %enchant_variation WEP_SOME_WEAPON shiny runed glowing randart
#    weapon_magic WEP_SOME_WEAPON_MAGIC
#
# You can also give a single variation mutliple tiles that are chosen from
# randomly when the item is created e.g. to have two randart tiles:
#
#    %enchant_variation WEP_SOME_WEAPON randart
#    weapon_randart1 WEP_SOME_WEAPON_RANDART
#    weapon_randart2
